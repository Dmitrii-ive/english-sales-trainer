# Sales English Trainer

Mobile-first PWA to drill English for discovery / sales calls. Content is generated by Claude from your meeting transcripts and pushed in via an API.

See [`README_FOR_CLAUDE.md`](README_FOR_CLAUDE.md) for the data contract used by future Claude sessions.

## Task types

1. **Sales phrases** — full expressions, 3 modes:
   - `flip` — show RU → reveal EN
   - `ru_en` — type the English translation
   - `audio_repeat` — TTS plays, you repeat
2. **Vocabulary** — single-word / collocation flashcards
3. **Speaking practice** — read EN aloud, browser TTS
4. **Roleplay** — customer objection → your spoken response (Phase 3, evaluated via Claude API)
5. **Cloze** — fill-in-the-blank from real meeting transcripts (Phase 2)
6. **Error finding** — rewrite a wrong sentence correctly (Phase 2)
7. **Quiz** — MCQ on product / methodology (Phase 2)

## Local setup

```bash
npm install

# 1. Generate secrets
openssl rand -hex 32   # → AUTH_SECRET
openssl rand -hex 32   # → ADMIN_TOKEN
npm run hash-password -- 'your-password'   # → APP_PASSWORD_HASH

# 2. Provision Vercel Postgres (after `vercel link`)
vercel env pull .env.local

# 3. Run migrations
npm run migrate

# 4. Dev server
npm run dev
```

Open `http://localhost:3000`, log in with your password.

## Architecture

- **Next.js 16** (App Router) + React 19 + Tailwind 4
- **Vercel Postgres** for data, accessed via `@vercel/postgres` template literals
- **Cookie auth**, single user, JWT signed with `AUTH_SECRET`
- **Admin API** (`/api/tasks`, `/api/weekly-plan`) gated by `ADMIN_TOKEN` — used by Claude to push content
- **Spaced repetition** via SM-2 (`src/lib/sr.ts`)
